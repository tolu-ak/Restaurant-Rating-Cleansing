2. Rename the file to california_restaurants_raw.csv
3. Place it in the root of the repository

# Restaurant-Rating-Cleansing
Which restaurant characteristics most strongly influence Yelp ratings, and can we develop a predictive model that estimates whether a restaurant will achieve a high customer rating based on attributes such as cuisine type, price range, location, and popularity?
